Instalacion

bajo una terminal como super usuario treclee los siguientes comandos :

    Instalar apache

    • apt-get install apache2

    Instalar Mysql

    • apt-get install mysql-server

    Instalar PHP5

    • apt-get install libapache2-mod-php5 php5 php5-mcrypt

    Iniciar el servidor

    • /etc/init.d restart,start,stop
    • service apache2 restart,start,stop
